Frequently Asked Questions

Why did the URL fetch fail?

Browsers have a security feature called CORS that prevents us from fetching other websites (like Google, Facebook) directly. If a URL fails, please open that page, press Ctrl+U to view source, copy the code, and paste it into the "Raw Code" tab.

Does CSS and JavaScript work?

Yes! Inline CSS (`<style>`) works perfectly. JavaScript also runs, but keep in mind that PDF is a static format, so animations won't play in the final file.

Is it safe?

Yes. Everything happens inside your browser's "Sandbox". No data is sent to our servers.
